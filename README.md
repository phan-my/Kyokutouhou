# Kyokutouhou
A Computer Science school project in Python

## Installation
### Debian
Prerequisites:
* `python3` ver. 3.11.2
* `pygame` ver. 2.1.2 and `pgzrun`
    * Run `sudo bash install-debian.sh`

Installation:
1. Run `git clone https://github.com/phan-my/kyokutouhou.git`
2. Run `python3 main.py`
